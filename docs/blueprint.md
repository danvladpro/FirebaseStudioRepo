# **App Name**: Excel Ninja

## Core Features:

- Shortcut Challenges: Presents a sequence of Excel shortcut challenges to the user. These challenges can vary from one time to another.
- Keypress Validation: Validates the user's keypresses against the required Excel shortcut for the current challenge, providing real-time feedback.
- Challenge Progression: Automatically advances to the next challenge upon successful completion of the current one.
- Performance Tracking: Tracks the user's time to complete each set of challenges and saves to local storage.
- Results Display: Presents the user's completion time and compares it against their best previous result for that set.
- Personalized Training Recommendations: Generates training recommendations by comparing performance on different types of shortcuts using a tool.
- Set Selection: Enables the user to select from a list of different Excel shortcut sets, each focusing on different categories of shortcuts.
- Welcome Page with User Stats: Displays user statistics such as fastest set completion time and last training date, along with other useful information.
- Challenge Sets: Programmed sets of 3 challenges each. User should see a list of different sets. Clicking on a set should bring the user to the challenge page.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey competence and precision.
- Background color: Light blue-gray (#F0F4F8) for a professional, uncluttered backdrop.
- Accent color: Yellow-orange (#FFAB40) to highlight interactive elements and important feedback messages.
- Body and headline font: 'Inter', a grotesque-style sans-serif, to give a machined, objective, neutral look.
- Use clean, minimalist icons to represent different sets and achievements.
- Maintain a clean, grid-based layout with ample whitespace for readability and focus.
- Subtle animations when a user completes an exercise or presses a button to provide feedback.
